¿Qué es el Phishing?

Es un método que los ciberdelincuentes utilizan para engañarte y obtener tu información confidencial como contraseñas o datos bancarios.

Cómo identificar un correo sospechoso:

Verifica el remitente: Asegúrate de que la dirección de correo sea oficial.

No compartas contraseñas: Ninguna empresa legítima te pedirá tu clave por correo.

Cuidado con los enlaces: Pasa el ratón sobre el enlace para ver la dirección real antes de hacer clic.
