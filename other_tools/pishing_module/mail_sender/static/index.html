3 Claves para detectar Phishing

Remitente Extraño

Desconfía si el dominio no coincide exactamente con el oficial (ej: @micompany.co en lugar de @micompany.com).

Sentido de Urgencia

Los atacantes suelen usar mensajes alarmantes como "Tu cuenta será bloqueada en 5 minutos" para forzarte a actuar rápido.

Enlaces Sospechosos
